# Pushing to Github from Jupyter

## Git in the terminal

### Step 1
Set up name & email in git by running 
$ git config --global user.email "you@example.com"
$ git config --global user.name "Your Name"

### Step 2
$ git clone https://github.com/OierGman/Learning_C.git
